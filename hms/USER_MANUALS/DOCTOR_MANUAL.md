## Overview

The Medicare HMS Doctor portal provides comprehensive tools for managing patient care, appointments, medical records, and clinical workflows. As a doctor, you have access to patient information, scheduling tools, and clinical documentation features.

### Key Features
- Patient information management
- Appointment scheduling and management
- Electronic medical records (EMR)
- Prescription management
- Lab test ordering and results
- Clinical documentation
- Patient communication tools

## Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Doctor credentials

### Login Process

1. **Access the System**
   - Navigate to: `http://your-hospital-domain.com/role-based-login`
   - Select "Doctor" role
   - Enter your credentials:
     - Email: Your assigned email address
     - Password: Your secure password

2. **First-Time Setup**
   - Review your profile information

   - Configure notification preferences
   - Familiarize yourself with the dashboard

The doctor dashboard provides:
- Today's appointments summary

- Prescription alerts

## Medical Records

### Clinical Documentation

**Progress Notes**
1. **Select Patient**
   - Navigate to patient profile
   - Click "Add Progress Note"

2. **Document Visit**
   ```
   Date: January 20, 2024
   Chief Complaint: Chest pain, shortness of breath
   History of Present Illness: 
     Patient reports chest discomfort for 2 days
     Pain described as pressure, 7/10 severity
     Shortness of breath with exertion
   
   Physical Examination:
     Vital signs: BP 140/90, HR 88, RR 18, Temp 98.6°F
     Cardiovascular: Regular rhythm, no murmur
     Respiratory: Clear breath sounds bilaterally
   
   Assessment:
     Hypertension with chest pain
     Possible angina
     Rule out cardiac emergency
   
   Plan:
     Order ECG and cardiac enzymes
     Prescribe nitroglycerin PRN
     Schedule stress test
     Follow up in 1 week
   ```

**SOAP Note Format**
- **S**ubjective: Patient's reported symptoms
- **O**bjective: Examination findings and measurements
- **A**ssessment: Clinical diagnosis and evaluation
- **P**lan: Treatment plan and follow-up

### Treatment Plans

**Creating Treatment Plans**

3. **Interventions**

**Monitoring Progress**

## Prescriptions

### Writing Prescriptions

1. **Select Patient**
   - Navigate to patient profile
   - Click "New Prescription"

2. **Medication Details**
   ```
   Medication: Lisinopril
   Dosage: 10mg
   Frequency: Once daily
   Route: Oral
   Duration: 30 days
   Quantity: 30 tablets
   Refills: 3
   Instructions: Take with food, monitor blood pressure
   ```

3. **Drug Interactions Check**
   - System automatically checks for interactions
   - Reviews current medications
   - Alerts for contraindications
   - Provides alternative suggestions

**Prescription Status**

**Refill Requests**

**Medication History**
- View all past prescriptions
- Track medication changes

## Lab Tests

### Ordering Lab Tests

1. **Select Tests**
   - Navigate to patient profile
   - Click "Order Lab Tests"
   - Choose from test catalog:
     ```
     Complete Blood Count (CBC)
     Comprehensive Metabolic Panel
     Lipid Profile
     HbA1c
     Thyroid Panel
     ```

2. **Test Details**
   ```
   Test: Complete Blood Count (CBC)
   Priority: Routine/Urgent/Stat
   Specimen: Blood
   Fasting Required: No
   Special Instructions: Draw in morning
   Clinical Indication: Routine checkup
   ```

3. **Submit Order**
   - Review test selection
   - Add clinical notes
   - Send to laboratory
   - Notify patient

### Reviewing Results

**Result Status**
- Pending: Test ordered, not yet completed
- In Progress: Sample being processed
- Completed: Results available
- Critical: Abnormal values requiring attention

**Interpreting Results**
1. **View Results**
   - Navigate to Lab Tests section
   - Select completed test
   - Review detailed results

2. **Compare with Reference Ranges**
   ```
   WBC: 7.2 K/µL (Normal: 4.5-11.0)
   RBC: 4.8 M/µL (Normal: 4.5-5.9)
   Hemoglobin: 14.5 g/dL (Normal: 13.5-17.5)
   Hematocrit: 43% (Normal: 41-50%)
   Platelets: 250 K/µL (Normal: 150-450)
   ```

3. **Add Interpretation**
   - Document clinical significance
   - Note abnormal values
   - Recommend follow-up actions
   - Update treatment plan

**Critical Values**
- Immediate notification for critical results
- Automatic alerts to doctor
- Patient notification protocols
- Documentation requirements
