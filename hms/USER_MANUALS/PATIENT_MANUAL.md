## Overview

The Medicare HMS Patient portal provides you with secure access to your personal health information, appointment scheduling, and communication with your healthcare providers. This user-friendly interface allows you to manage your healthcare journey from the comfort of your home.

### Key Features
- View and update personal information
- Schedule and manage appointments
- Access medical records and test results
- View and pay bills online
- Request prescription refills
- Communicate with healthcare providers
- Receive health reminders and notifications

## Getting Started

### System Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Patient account credentials

### Login Process

1. **Access the System**
   - Navigate to: `http://your-hospital-domain.com/role-based-login`
   - Select "Patient" role
   - Enter your credentials:
     - Email: Your registered email address
     - Password: Your secure password

2. **First-Time Login**
   - Review and accept terms of service
   - Set up security questions
   - Configure notification preferences
   - Complete profile information

The patient dashboard provides:

- Quick access to common tasks

## Appointments

### Scheduling New Appointments

1. **Book Appointment**
   - Click "Appointments" in the main menu
   - Select "Book New Appointment"

2. **Choose Department/Doctor**
   ```
   Department: Cardiology
   Doctor: Dr. Sarah Johnson
   Specialization: Heart and cardiovascular care
   ```

3. **Select Date and Time**
   - View available dates in calendar
   - Choose preferred time slot
   - Consider appointment type:
     - Consultation (30 minutes)
     - Follow-up (15 minutes)
     - Procedure (varies)

4. **Provide Reason for Visit**
   ```
   Reason: Chest pain follow-up
   Symptoms: Mild chest discomfort, shortness of breath
   Urgency: Routine/Urgent/Emergency
   ```

5. **Confirm Appointment**
   - Review appointment details
   - Set reminder preferences
   - Confirm booking
   - Receive confirmation email/SMS

### Managing Appointments

**Viewing Appointments**
- Upcoming appointments list
- Calendar view for monthly overview
- Appointment details and status
- Preparation instructions

**Rescheduling Appointments**
1. Select appointment to reschedule
2. Choose new date and time
3. Confirm changes
4. Update calendar
5. Receive new confirmation

**Cancelling Appointments**
1. Select appointment to cancel
2. Provide cancellation reason
3. Confirm cancellation
4. Receive cancellation confirmation
5. Note cancellation policy

**Appointment Types**
- **Consultation**: Initial or routine visits
- **Follow-up**: Ongoing care visits
- **Procedure**: Medical procedures or tests
- **Emergency**: Urgent medical attention
- **Telemedicine**: Virtual consultations

**Normal Ranges**
- Green: Within normal range
- Yellow: Borderline/monitor
- Red: Abnormal/attention needed
- Gray: Insufficient data

**Result Interpretation**
- Compare with previous results
- Look for trends over time
- Note significant changes
- Discuss with your doctor

## Communication

### Messaging Your Providers

**Compose Message**
1. Navigate to "Messages"
2. Click "New Message"
3. Select recipient:
   - Primary care physician
   - Specialist
   - Nurse practitioner
   - Administrative staff

4. **Message Content**
   ```
   Subject: Medication Question
   Message: I have a question about my new prescription...
   Priority: Routine/Urgent
   Attachments: None
   ```

**Message Types**
- Medical questions
- Appointment requests
- Prescription inquiries
- Billing questions
- General inquiries

### Receiving Messages

**Message Categories**
- **Clinical**: Medical-related messages
- **Administrative**: Appointments, billing
- **System**: Notifications, reminders
- **Emergency**: Urgent medical alerts

**Response Times**

5. Go to nearest emergency room
